Create k8s POC cluster

We use Kind(Kubernetes in docker) to help create k8s cluster.
save below file as AspiraSCM-POC.yaml , please update IP address 172.28.24.67 with your host's IP.

kind: Cluster
apiVersion: kind.sigs.k8s.io/v1alpha3
networking:
  apiServerPort: 1900
  apiServerAddress: 172.28.24.67

kubeadmConfigPatchesJson6902:
- group: kubeadm.k8s.io
  version: v1beta2
  kind: ClusterConfiguration
  patch: |
    - op: add
      path: /apiServer/certSANs/-
      value: 172.28.24.67
# 1 control plane node and 3 workers
nodes:
# the control plane node config
- role: control-plane
# the three workers
- role: worker
- role: worker
- role: worker

following (https://kind.sigs.k8s.io/docs/user/quick-start/) to download kind.
create a k8s cluster

# kind create cluster --name AspiraSCM-POC --config ./AspiraSCM-POC.yaml --image kindest/node:v1.16.2
